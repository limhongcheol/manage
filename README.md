# 메이플스토리 길드 관리 콘솔 프로그램 (C / Windows)

본 프로젝트는 **메이플스토리 길드 운영을 보조하기 위한 Windows 콘솔 기반 관리 프로그램**으로,
길드원 정보 관리부터 **수로**, **플래그 점수**, **길드 스킬 사용 가능 인원**, **주차별 통계**까지 한 번에 관리할 수 있도록 구현되었습니다.
---

## 1. 주요 기능

### 1) 길드원 관리

* 길드원 **신규 등록 / 수정 / 삭제**
* 닉네임, 레벨, 직업 정보 관리
* `maple.gg` 페이지 복사·붙여넣기를 통한 **대량 등록 지원**
* 길드 탈퇴자 자동 탐지 및 일괄/개별 삭제

### 2) 수로 관리

* 날짜 기반 수로 참여 기록 저장 (최대 100회)
* 특정 길드원의 수로 참여 이력 조회
* **주차 계산 로직**을 통한 회차 자동 산출
* 주차별 수로 참여 인원 통계 및 시각화(텍스트 바)

### 3) 플래그 점수 관리

* 길드원별 주간 플래그 점수 입력
* 주간 플래그 → 누적 플래그 자동 이관
* 일주일 경과 시 플래그 자동 초기화
* 주간 플래그 랭킹 및 참여율 계산

### 4) 길드 스킬 사용 가능 여부 판단

* 최근 2~3주 수로 참여 여부 + 플래그 점수 기준 판별
* **길드 스킬 사용 가능 / 불가능 인원 자동 분류**
* 컬러 출력으로 가독성 강화

### 5) 통계 기능

* 캐릭터별 수로 참여 횟수 통계
* 날짜(회차) 기준 수로 참여 인원 통계
* 주간 플래그 상위 랭킹 출력

* **OS**: Windows
* **언어**: C (Visual Studio 기준)


* 콘솔 색상 및 커서 제어를 위해 `windows.h` 사용

## 3. 데이터 구조

### 길드원 구조체

```c
struct charcter_info {
    char* name;              // 닉네임
    int level;               // 레벨
    char* job;               // 직업
    char goday[100][11];     // 수로 참여 날짜 (YYYY/MM/DD)
    int twoweeks_pla;        // 이번주 플래그 점수
    int total_pla;           // 누적 플래그 점수
    int deleted;             // 활성/삭제 여부
    int temporarly;          // 임시 상태 (스킬 판별용)
    int checkout;            // 탈퇴 확인용
};
```

* 최대 길드원 수: **200명**
* 수로 기록 최대: **100회**

---

## 4. 파일 입출력

### 저장 파일

* 파일명: `guild_data`
* 프로그램 종료 시 자동 저장
* 다음 실행 시 자동 로드

### 저장 정보

* 전체 길드원 수
* 마지막 수정 날짜
* 각 길드원의 기본 정보 + 수로 기록

---

## 5. 메뉴 구성

```
1. 길드원 전체 조회
2. 길드원 신규 등록
3. 기존 길드원 삭제
4. 기존 길드원 정보 수정
5. 수로 정보 입력
6. 플래그 점수 입력
7. 길드 스킬 사용 가능 인원 확인
8. 프로그램 종료 (저장)
9. 길드 탈퇴자 확인
10. 주간 플래그 순위
11. 통계
```
